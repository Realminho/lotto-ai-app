lotto_ai_app_v9 업데이트 방법

1. 이 압축파일을 풉니다.
2. GitHub 저장소에서 기존 app.py, requirements.txt를 이 파일들로 교체합니다.
3. Commit changes를 누릅니다.
4. Streamlit Cloud 앱이 자동 반영되지 않으면 Manage app > Reboot를 누릅니다.

v9 주요 변경점
- 전부 홀수, 전부 짝수, 전부 저번호, 전부 고번호 같은 극단 패턴을 후보에서 배제하지 않습니다.
- 특정 구간/십대/끝자리 쏠림 패턴도 과거 등장빈도로 따로 분석합니다.
- 홀짝/저고 2개만 보던 방식에서 합계, 표준편차, 범위폭, 구간분포, 십대분포, 소수/합성수, 배수, 나머지, 연속번호, 간격, 직전 회차 재등장 수까지 확장했습니다.
- 추천 조합 생성 시 상위 통계번호뿐 아니라 전체 1~45, 과거 실전 패턴 복원, 극단/쏠림 후보를 함께 생성합니다.
- 조합 점수에서 단순 균형 패널티를 줄이고 실제 과거 패턴 빈도를 더 강하게 반영합니다.

주의
- 로또는 독립 무작위 추첨이므로 어떤 통계도 당첨을 보장하지 않습니다.
- mirror 데이터는 비공식 데이터이므로 최신 회차는 동행복권 공식 사이트에서 최종 확인하세요.
