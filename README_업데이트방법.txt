lotto_ai_app_v10 업데이트 방법

1. 이 압축파일을 풉니다.
2. GitHub 저장소에서 기존 app.py, requirements.txt를 이 파일들로 교체합니다.
3. Commit changes를 누릅니다.
4. Streamlit Cloud 앱이 자동 반영되지 않으면 Manage app > Reboot를 누릅니다.

v10 주요 변경점
- 직전 회차 본번호가 다음 회차 본번호로 몇 개 다시 나왔는지 패턴화했습니다.
- 직전 회차 보너스 번호가 다음 회차 본번호로 다시 나오는 경우를 별도 패턴으로 분석합니다.
- 직전 회차 본번호/보너스가 다음 회차 보너스로 다시 나오는 경우도 패턴 분석 탭에 포함했습니다.
- 최근 2회, 3회, 5회, 10회 범위에서 본번호/보너스가 다시 등장하는 개수를 모두 빈도표로 계산합니다.
- 추천 조합 점수에 직전/최근 재등장 패턴 점수를 반영합니다.
- 추천 후보 생성 단계에서 직전 전체 7개 번호 일부를 포함하는 후보와 직전 보너스 번호를 포함하는 후보를 강제로 생성해, 재등장 가능성을 처음부터 배제하지 않게 했습니다.
- 추천 조합 표와 조합별 수치 요약에 직전본번호재등장, 직전보너스본번호재등장, 직전전체7개재등장, 최근5회본번호재등장, 최근5회보너스본번호재등장 항목을 추가했습니다.

주의
- 로또는 독립 무작위 추첨이므로 어떤 통계도 당첨을 보장하지 않습니다.
- mirror 데이터는 비공식 데이터이므로 최신 회차는 동행복권 공식 사이트에서 최종 확인하세요.
